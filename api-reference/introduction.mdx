## Base URL

All API requests should be made to:

```
https://api.pikzels.com
```

## Authentication

All API endpoints require an API key in the `X-Api-Key` header:

```bash
curl -H "X-Api-Key: pkz_your_api_key_here" https://api.pikzels.com/v1/thumbnail
```

**Get your API key:**
1. Sign up at [app.pikzels.com/platform](https://app.pikzels.com/platform)
2. Navigate to API Keys section
3. Generate a new key (starts with `pkz_`)

**Security:** Keep your API key secure - never expose it in client-side code or commit to version control.

## Response Format

All endpoints return JSON responses with the following structure:

### Success Response
```json
{
  "output": "https://media.pikzels.com/..."
}
```

### Error Response
```json
{
  "error": {
    "code": "ERROR_CODE",
    "message": "Human-readable error message",
    "details": "Additional error details"
  }
}
```

## Common Error Codes

- `400` - Bad Request: Invalid or missing fields
- `401` - Unauthorized: Invalid authentication
- `402` - Payment Required: Insufficient balance
- `429` - Too Many Requests: Rate limit exceeded
- `500` - Internal Server Error
- `503` - Service Unavailable: Maintenance mode
