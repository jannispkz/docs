<ParamField query="id" type="string" required>
  The persona ID returned from the training request
</ParamField>

## Status Values

<AccordionGroup>
  <Accordion title="Queued" icon="clock">
    Training request is in the queue and will begin processing soon.
  </Accordion>
  
  <Accordion title="Analyzing" icon="magnifying-glass">
    Images are being analyzed and prepared for training.
  </Accordion>
  
  <Accordion title="Training" icon="brain">
    The AI model is actively learning from your images. This can take up to 3 minutes.
  </Accordion>
  
  <Accordion title="Ready" icon="check">
    Training is complete! Your persona is ready to use in generation requests.
  </Accordion>
</AccordionGroup>

<Warning>
The `temp_portrait_url` is only available for 12 hours after training completes. Download and host this image yourself if you want to keep it permanently.
</Warning>

<RequestExample>
```bash Example Request
curl --location --request GET 'https://api.pikzels.com/v1/persona?id=abc123def456' \
--header 'X-Api-Key: YOUR_API_KEY'
```
</RequestExample>

<ResponseExample>
```json Training Complete
{
  "id": "abc123def456",
  "created": 1704067200,
  "name": "My Character",
  "status": "Ready",
  "progress": 100,
  "temp_portrait_url": "https://media.pikzels.com/temp/portrait_abc123.jpg"
}
```

```json Still Training
{
  "id": "abc123def456", 
  "created": 1704067200,
  "name": "My Character",
  "status": "Training",
  "progress": 65,
  "temp_portrait_url": "none"
}
```
</ResponseExample>

## Using Your Persona

Once status is "Ready", use the persona ID in generation requests:

<CodeGroup>
```json Thumbnail Generation
{
  "prompt": "epic gaming moment",
  "format": "16:9",
  "model": "pkz-3", 
  "persona": "abc123def456"
}
```

```json Face Swap
{
  "image_url": "https://example.com/target.jpg",
  "face_url": "https://example.com/source.jpg",
  "persona": "abc123def456"
}
```
</CodeGroup>
