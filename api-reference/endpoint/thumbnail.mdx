---
title: "Thumbnail"
api: "POST https://api.pikzels.com/v1/thumbnail"
description: "Create a new thumbnail image based on a text prompt"
---

import CommonErrors from '/snippets/common-errors.mdx';

<ParamField body="prompt" type="string" required>
  Text prompt describing the desired thumbnail
</ParamField>

<ParamField body="format" type="string" required>
  Aspect ratio format. Options: `16:9`, `9:16`, `1:1`
</ParamField>

<ParamField body="model" type="string" required>
  AI model to use for generation. Options: `pkz-1`, `pkz-2`, `pkz-3`
</ParamField>

<ParamField body="persona" type="string">
  Optional persona ID from [custom training](/api-reference/endpoint/persona-create) for consistent character generation
</ParamField>

<ParamField body="style" type="string">
  Optional style ID from [custom training](/api-reference/endpoint/style-create) to apply specific artistic styles
</ParamField>

<RequestExample>
```bash Example Request
curl --location --request POST 'https://api.pikzels.com/v1/thumbnail' \
--header 'X-Api-Key: YOUR_API_KEY' \
--header 'Content-Type: application/json' \
--data-raw '{
  "prompt": "cat",
  "format": "16:9",
  "model": "pkz-2"
}'
```

```bash With Custom Training
curl --location --request POST 'https://api.pikzels.com/v1/thumbnail' \
--header 'X-Api-Key: YOUR_API_KEY' \
--header 'Content-Type: application/json' \
--data-raw '{
  "prompt": "epic gaming moment",
  "format": "16:9",
  "model": "pkz-3",
  "persona": "abc123def456",
  "style": "xyz789ghi012"
}'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "output": "https://media.pikzels.com/temp/pikzels_1234567890.jpeg"
}
```
</ResponseExample>
