---
title: "Recreate"
api: "POST https://api.pikzels.com/v1/recreate"
description: "Recreate an image with AI enhancement and modifications"
---

<ParamField body="image_url" type="string" required>
  URL of the source image to recreate
</ParamField>

<ParamField body="model" type="string" required>
  AI model to use for recreation. Options: `pkz-2`, `pkz-3`
</ParamField>

<ParamField body="image_weight" type="string" default="medium">
  How closely to follow the original image. Options: `low`, `medium`, `high`
</ParamField>

<ParamField body="prompt" type="string">
  Optional prompt to guide the recreation. Changes work best on pkz-3
</ParamField>

<ParamField body="persona" type="string">
  Persona ID that you received from training
</ParamField>

<ParamField body="style" type="string">
  Style ID that you received from training
</ParamField>

<RequestExample>
```bash Example Request
curl --location --request POST 'https://api.pikzels.com/v1/recreate' \
--header 'X-Api-Key: YOUR_API_KEY' \
--header 'Content-Type: application/json' \
--data-raw '{
  "image_url": "https://images.unsplash.com/photo-1506748686214-e9df14d4d9d0",
  "model": "pkz-2",
  "image_weight": "medium"
}'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "output": "https://media.pikzels.com/temp/recreated_5f6e7d8c9b0a.jpeg"
}
```
</ResponseExample>
