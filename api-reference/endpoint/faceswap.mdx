---
title: "FaceSwap"
api: "POST https://api.pikzels.com/v1/faceswap"
description: "Seamlessly swap faces between images"
---

import CommonErrors from '/snippets/common-errors.mdx';

<ParamField body="image_url" type="string" required>
  URL of the target image where face will be placed
</ParamField>

<ParamField body="face_url" type="string" required>
  URL of the source face image
</ParamField>

<ParamField body="persona" type="string">
  Optional persona ID from [custom training](/api-reference/endpoint/persona-create). Use this instead of face_url to swap with your trained persona
</ParamField>

<RequestExample>
```bash Example Request
curl --location --request POST 'https://api.pikzels.com/v1/faceswap' \
--header 'X-Api-Key: YOUR_API_KEY' \
--header 'Content-Type: application/json' \
--data-raw '{
  "image_url": "https://images.unsplash.com/photo-1507003211169-0a1dd7228f2d",
  "face_url": "https://images.unsplash.com/photo-1438761681033-6461ffad8d80"
}'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "output": "https://media.pikzels.com/temp/faceswap_3e4f5a6b7c8d.jpeg"
}
```
</ResponseExample>

## Common Errors

<CommonErrors />

### Endpoint-Specific Errors

| Status Code | Code | Description |
|-------------|------|-------------|
| 400 | `INVALID_ORIGINAL_IMAGE` | Target image is missing, corrupted, or contains no face |
| 400 | `INVALID_FACE_IMAGE` | Source face image is missing, corrupted, or contains no face |
