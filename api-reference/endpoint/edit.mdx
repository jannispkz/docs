---
title: "Edit Image"
api: "POST https://api.pikzels.com/v1/edit"
description: "Edit an existing image using AI-powered modifications"
---

<ParamField body="prompt" type="string" required>
  Instructions for how to edit the image
</ParamField>

<ParamField body="image_url" type="string" required>
  URL of the source image to edit
</ParamField>

<ParamField body="mask_url" type="string">
  Optional mask URL for selective editing
</ParamField>

<RequestExample>
```bash Example Request
curl --location --request POST 'https://api.pikzels.com/v1/edit' \
--header 'X-Api-Key: YOUR_API_KEY' \
--header 'Content-Type: application/json' \
--data-raw '{
  "prompt": "make the cat blue",
  "image_url": "https://example.com/cat.jpg"
}'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "output": "https://media.pikzels.com/temp/edited_1234567890.jpeg"
}
```
</ResponseExample>
