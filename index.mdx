## Simple Integration

Getting started is straightforward. Here's a quick example:

```bash
curl -X POST https://api.pikzels.com/v1/thumbnail \
  -H "X-Api-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "TOP 10 space facts with astronaut helmet",
    "format": "16:9",
    "model": "pkz-2"
  }'
```
